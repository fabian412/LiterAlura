Primer readme del challengue de alura, LiterAlura. En este challengue se usara una api, para construir un catalogo de libros.

El desafio fue muy grande, pero gracias a codigos subidos en el discord de alura latam, pude resolver la mayoria de dudas generadas.

el programa consiste en utilizar una api, en este caso gutendex para que el usuario pueda buscar libros, estos se van guardando en una base de datos SQL, el principal desafio fue
el formato de busqueda, ya que la api, es principalmente para buscar autores, y el desafio pedia libros, por ende se debe transformar y extraer esta informacion, para que sea asquible, 
a lo pedido por el usuario.

el programa tiene las siguientes opciones:
Buscar libros, donde buscara libros en la api gutendex y retornara informacion respecto al libro.
Listar libros regristrados, en donde se mostraran los diferentes libros buscados con anterioridad.
Listar libros por idioma, donde el usuario colocara un idioma, y mostrara los libros en ese respectivo idioma.
Listar autores registrados, donde se mostraran los autores guardados en la base de datos SQL.
Listar 10 libros mas descargados, donde se mostraran los libros mas descargados.
Buscar autores por nombres, donde enves de buscar los libros, aparacera los autores con sus diferentes libros escritos.
